# LScript Syntax Highlighting

Adds syntax highlighting for the **LScript** language.

### Features
- Highlights keywords, strings, numbers, comments, functions, and classes
- Supports contextual keywords like `force` (only after `import`) and `until` (only in `loop`)

### Usage
1. Copy this folder to your VSCode extensions directory, or run `vsce package` and install the `.vsix`.
2. Create a file with extension `.mylang`.
3. Start coding!
